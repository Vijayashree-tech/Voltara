⚡ Voltara – AI-Powered Smart Energy Management Platform

📌 Overview

Voltara is an AI-powered energy intelligence platform designed to help households monitor electricity consumption, estimate future energy usage, calculate electricity bills, track carbon emissions, and receive personalized energy-saving recommendations.

The platform combines Machine Learning, Sustainability Analytics, Carbon Tracking, and Smart Recommendation Systems to provide actionable insights for reducing energy consumption and promoting sustainable living.

🚀 Features
⚡ Electricity Consumption Prediction

Predicts household energy consumption using a trained Machine Learning model.

💰 Smart Bill Estimation

Calculates estimated monthly electricity bills based on predicted energy usage.

🌍 Carbon Footprint Tracking

Estimates carbon emissions generated from household electricity consumption.

🏆 Sustainability Score

Provides a sustainability score that evaluates energy efficiency.

🚨 Smart Energy Alerts

Detects abnormal usage patterns and notifies users.

💡 AI Recommendations

Generates personalized recommendations to reduce electricity consumption and costs.

🔋 Appliance Analytics

Provides appliance-level energy distribution insights.

📈 Energy Forecast Dashboard

Visualizes predicted future energy trends and usage patterns.

🧠 Model Training
Install Dependencies
pip install pandas scikit-learn joblib
Train Model
python train_model.py

After training, the model file will be generated:
electricity_bill_model.pkl

⚙️ Backend Setup
Navigate to Backend
cd backend
Create Virtual Environment
Windows
python -m venv venv

Activate:

venv\Scripts\activate
Linux / macOS
python3 -m venv venv
source venv/bin/activate
Install Dependencies
pip install -r requirements.txt
Run Backend
python app.py

🎨 Frontend Setup
Navigate to Frontend
cd frontend
Install Dependencies
npm install
Run Frontend
npm run dev
